    요 근래 아침 일찍 일어나는 것을 꾸준히 기록하는 것에 실패했다. 왜일까?

    일단 내가 원하는 것은 최종적으로 체중감량인데, 이를 위해서는 꾸준한 헬스장 출석을 해야한다. 헬스장에 출석하기 위해서는 일찍 일어나야한다. 일찍 일어나기 위해서는 일찍 자야한다. 일찍 잠에 들기 위해서는 내 수면패턴을 완벽하게 파악하고 수면에 방해되는 것들을 제거한다. 수면에 방해되는 것은 대표적으로 유튜브앱/웹툰앱이다. 이는 지키자 앱을 통해서 접근 제어를 하면서 해결이 되었다. 간혹 발생하는 야근의 경우, 하루 이틀이라 루틴을 깨는 정도까지는 아닌 것 같다. 수면 패턴을 파악하기 위해서는 꾸준한 기록이 필요하다. 그래서 지금 요 글을 통해 꾸준한 기록을 해보려고 했으나 꾸준한 기록을 실패했다.

    꾸준한 기록을 내 루틴으로 가져가야한다. 내가 하루 중 빠짐없이 반드시 하는 것은 무엇일까? 그리고 그것과 엮어서 반드시 루틴을 챙기고 싶다.

    내가 매일 하는 것: 샤워 / 머리말리기 / (평일) 회사 맥북 켜기 혹은 (주말) 개인 맥북 켜기


    맥북을 켜고 바로 화면이 뜨는게 있으면 매우 좋을 것 같다. (안하면 안넘어가게..)


    오 Shortery + 단축어 + shell 를 통해 해결했다. 이제 이걸 매일 하루 한번 동작되게 해야한다.


    ```

    today=$(date +%Y-%m-%d) # get today's date in YYYY-MM-DD format

    if [[ $(cat TODAY) == $today ]]; then # check if TODAY file content is equal to today's date

    exit 0 # skip if they are equal

    fi


    echo $today > TODAY # write today's date to TODAY file
